## Replication Code and Analysis
The [/data](https://github.com/nadiyakostyuk/National-Cybersecurity-Strategy-Data/tree/main/data) folder contains the data used for the analysis.

The [/code](https://github.com/nadiyakostyuk/National-Cybersecurity-Strategy-Data/tree/main/code) folder contains the code used to produce the results.

The [/output](https://github.com/nadiyakostyuk/National-Cybersecurity-Strategy-Data/tree/main/output) folder contains the output for the main results.
